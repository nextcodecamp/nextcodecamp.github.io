NLTK

NLTK เป็นไลบรารีที่พัฒนาขึ้นด้วยภาษา Python สำหรับการประมวลผลภาษาธรรมชาติ ใน NLTK ประกอบด้วยโมดูลและไลบรารี่จำนวนมากที่สามารถทำ Natural Processing Language ได้ Natural Processing Language หรือ NLP คือ การประมวลผลภาษาธรรมชาติ ภาษาธรรมชาติคือภาษาที่สื่อสารด้วยมนุษย์ การประมวลผลคือการนำคอมพิวเตอร์เข้ามาใช้เพื่อทำการจัดการกับข้อความที่ได้จากภาษาธรรมชาติ เพื่อทำให้คอมพิวเตอร์สามารถเข้าใจภาษาธรรมชาติได้

การเรียกใช้ NLTK
ให้ทำการ import nltk โดยในการสอนนี้จะทำการกับ Grammar ดังนั้นจึงเรียกใช้ CFG (Context Free Grammar) 

import nltk
from nltk import CFG

เรียกใช้ nltk.parse.generate ซึ่งเป็นไลบรารี่ที่ทำงานในการสร้างประโยคภาษาธรรมชาติที่สร้างขึ้นจาก Grammar

from nltk.parse.generate import generate, demo_grammar
ตัวอย่างการเรียกใช้ไวยากรณ์ (grammar) ที่เป็นข้อมูลทดลอง (demo grammar)การใช้คำสั่ง for สามารถวนลูปเพื่อแสดงผลลัพธ์ที่ parser ทำงานในการสร้างประโยค

grammar = CFG.fromstring(demo_grammar)
print(grammar)
print("Sentence generated by the NLTK parser")
for sentence in generate(grammar, depth=4, n=20):
   print(' '.join(sentence))

ตัวอย่างการสร้างไวยากรณ์ซึ่งนำเข้าสู่ CFG 
การสร้างไวยากรณ์จะประกอบด้วย Productions หรือประพจน์ซึ่งประกอบด้วย Nonterminal symbol และ Terminal symbol
- Nonterminal symbol ใน CFG ซึ่งหมายถึงข้อความที่มีนิยามที่เกี่ยวข้องและสามารถนำไปสร้างประโยคได้ แสดงด้วยคำที่ใช้ภาษาอังกฤษตัวพิมพ์ใหญ่หรือเล็กก็ได้เช่น S, NP, VP เป็นต้น
- Terminal symbol คือข้อความที่สิ้นสุด หรือแสดง lexical (ชิ้นส่วนในไวยากรณ์) แสดงด้วยคำที่ใช้ภาษาอังกฤษตัวพิมพ์ใหย๋หรือเล็กก็ได้ แต่ต้องกำหนดใน Single quote  เช่น 'a', 'the', 'dog', 'cat', 'sat' เป็นต้น
ตัวอย่างการสร้างไวยากรณ์

grammar = CFG.fromstring("""
S -> NP VP
PP -> P NP
NP -> Det N | NP PP
VP -> V NP | VP PP
Det -> 'a' | 'the'
N -> 'dog' | 'cat'
V -> 'chased' | 'sat'
P -> 'on' | 'in'
""")

เมื่อต้องการสร้างประโยคสามารถกำหนดด้วยคำสั่ง

for sentence in generate(grammar, n=10):
    print(' '.join(sentence))
คำสั่งอื่นๆที่สามารถใช้กับ NLTK ได้
- grammar 
เพื่อแสดงจำนวน Production ในไวยากรณ์ที่กำหนด ซึ่ง Production 
เสมือนกฎในการสร้างประโยคต่างๆในไวยากรณ์ตำอย่างเช่น 

S -> NP VP 
PP -> P NP 
NP -> Det N | NP PP 

จาก S->NP VP หมายความว่า S คือประโยคที่ประกอบด้วยคำนามวลี (NP) และ คำกิริยา (VP) สัญลักษณ์ | แสดงทางเลือกในการสร้างประโยค เช่น NP -> Det N | NP PP แสดงทางเลือกในการสร้างประโยคที่ประกอบด้วยคำขยาย (Det) และคำนาม(N) หรือ นาม (NP) และ บรุพบท (PP)

- grammar.start() 
เพื่อแสดง Start symbol โดย Start symbol เป็น Nonterminal ที่เป็นจุดเริ่มต้นในการสร้างประโยคทุกๆไวยากรณ์จะต้องกำหนด Start symbol ซึ่งจะกำหนดเพียง 1 Start symbol เท่านั้น

- grammar.productions() 
แสดงทุกๆ Production ในไวยากรณ์

- list(generate(grammar,n=10))
แสดงประโยคที่สร้างได้จากไวยากรณ์

nltk.parse

Import library
 
import nltk
from nltk import CFG
from nltk.parse.generate import generate

สร้างไวยากรณ์

grammar = CFG.fromstring("""
S -> Sub Verb Object
Sub -> 'ฉัน' | 'ผม' | 'เรา'
Verb -> 'รัก'| 'ไม่ชอบ'
Object -> 'คุณ'|'เขา'
""")

สร้างประโยคprint("Sentence generated by the NLTK parser")

for sentence in generate(grammar,depth=3,n=10):      
    print(' '.join(sentence))

กำหนดประโยคที่ต้องการให้ Parser นำไปสร้าง Parse tree

terminalsymbol = ['เรา','รัก','คุณ']
parser = nltk.ChartParser(grammar)
for tree in parser.parse(terminalsymbol):
   print(tree)
   tree.pretty_print(unicodelines=True, nodedist=3)
