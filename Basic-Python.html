EP.1 เริ่มต้นไพธอนด้วยการเขียนข้อความกับคำสั่ง print

ตัวอย่างที่ 1 การใช้คำสั่ง print ซึ่งกำกับด้วย Single quote หรือ Double quote
print("Hello")
print('Hello')

ตัวอย่างที่ 2 แสดงข้อความจากตัวแปร 
text = "Basic python" #text มีชนิดข้อมูล เป็น String
text2 = "This is the EP1."
print("Message ",text,text2)
print("Message is ",text," example ",text2)
print("First message is ",text, "\n Second message is ", text2)

ตัวอย่างที่ 3 การแสดงข้อความที่มีสัญลักษณ์พิเศษ (เช่น " , ', #, @)
print("The message has double quote \"\" ")
print('The message has double quote \'\' ')
print("Copyright @2022 ")
print('Copyright @2022 ')

EP.2 ชนิดข้อมูล boolean, int, float 

Data type

ชนิดข้อมูลพื้นฐานใน Python
- Boolean type
- Numeric type - int, float, complex
- Sequence Types — list, tuple, range
- Text Sequence Type — str
- Mapping Type - dict

Boolean type
ชนิดข้อมูลบูลีน มีค่าที่เป็นไปได้คือ จริง (True) หรือ เท็จ (False)
clockStart = True
print("Status of clock is ",clockStart)
clockStart = False
print("Status of clock is ",clockStart)

Numeric type
ตัวอย่างที่ 1 ชนิดข้อมูล int แสดงเลขจำนวนเต็มซึ่งเป็นบวกหรือลบก็ได้
price = 100
discount = -20
total = price + discount # total = price - discount if discount is negative value.
print("Product price is ",price)
print("Discount is ",discount)
print("Total product price is ",total)

ตัวอย่างที่ 2 ชนิดข้อมูล float แสดงเลขจำนวนจริงหรือเลขที่มีจุดทศนิยม
savingMoney = 500000
interestRate = 0.05
savingInterest = 500000 * interestRate
print("I have ",savingMoney," in the bank account.")
print("Interest rate is ",interestRate)
print("I will get interest ",savingInterest) 

EP.3 ชนิดข้อมูลแบบลิสต์ (List)

Sequence type 
List
ตัวอย่างที่ 1 การสร้างลิสต์ด้วยการประกาศค่า
day = ["Monday" ,"Tuesday" ,"Wednesday" ,"Thursday","Friday","Saturday","Sunday" ] # สมาชิกจะมีชนิดข้อมูลแบบเดียวกัน เรียกว่า Homogenous data
print("All days are ",day)
print("First day of the week is ",day[0])
print("Last day of the week is ",day[6])

ตัวอย่างที่ 2 การสร้างลิสต์ด้วยการเพิ่มสมาชิกเข้าลิสต์
weekend = []
weekend.append("Saturday")
weekend.append("Sunday")
print(weekend)
len(weekend) #คำสั่ง len หาจำนวนสมาชิกที่อยู่ลิสต์ได้

ตัวอย่างที่ 3 การแก้ไขค่าและลบค่าในลิสต์
day.remove("Sunday")
print("After remove:",day)
day.append("Sunday")
print("After append:",day)

แก้ไขค่าในลิสต์
day[0]="23 Monday"
print("Add 23 to Monday",day[0])
day[0]="Monday"
print("Create day",day[0])
print(day)


เอาสมาขิกออกจากลิสต์ แล้วสังเกตดูการอ้างอิง
print("Wednesday is indexed by 2 ")
day.remove("Wednesday")
print(day)
print("Member indexed by 2 is ",day[2])
day.insert(2,"Wednesday")
print(day)

ตัวอย่างที่ 4 การเรียกใช้ reverse ในลิสต์
print(weekend)
weekend.reverse()
print(weekend)

EP.4 ชนิดข้อมูล Tuple และการหาชนิดข้อมูลของตัวแปร

Tuple
เป็นชนิดข้อมูลที่ไม่สามารถเปลี่ยนแปลงค่าได้ เป็นชนิดข้อมูลที่มีสมาชิกที่มีชนิดข้อมูลต่างกันได้

ตัวอย่างที่ 1 การสร้าง Tuple
mytuple = ("Adele", 23, "Female", 160)
print(mytuple)
print("Name:",mytuple[0])
print("Age:",mytuple[1])
print("Sex:",mytuple[2])
print("Height:",mytuple[3])

ตัวอย่างที่ 2 การนับสมาชิกใน Tuple
mytuple.count("Female")

ตัวอย่างที่ 3 การสร้างลิสต์ใน tuple 
student = (["Nana", 23, "Female", 160],
["Alex",25,"Male",170],
["Peter", 23,"Male",175])
print("Student 1:", student[0])
print("Student 2:", student[1])
print("Student 3:", student[2])
print("There are ",len(student)," persons.")

การหาชนิดข้อมูลของตัวแปรสามารถใช้คำสั่ง type(...)
myvariable = 300                 #Dynamic type binding เปลี่ยนแปลงชนิดข้อมูลได้ตลอดเวลารัน
type(myvariable)
myvariable = 44.556
type(myvariable)
myvariable = "World"
type(myvariable)

EP.5 ชนิดข้อมูล String และการสร้างลิสต์จากสตริง

String
เราสามารถเขียนสตริงใน 2 รูปแบบ คือการแสดงข้อความสตริงบรรทัดเดียว และแบบหลายบรรทัด

ตัวอย่างที่ 1 การแสดงข้อความบรรทัดเดียวด้วย Double quote และ Single
str1 = "Friday."
str2 = "last day of the week."
print(str1," is a ",str2)

ตัวอย่างที่ 2 การแสดงข้อความแบบหลายบรรทัดกำกับด้วย """....""" หรือ '''....'''
multiline = '''
Hello!
'Today is Friday'. @24March,2022
'''
print("Multiline string:",multiline)

ตัวอย่างที่ 3 หาความยาวสตริงด้วยฟังก์ชัน len
message = "Hello Friday"
print("Length of ",message," is ", len(message))

ตัวอย่างที่ 4 การเข้าถึงอักขระในสตริง
print(message, " has ",message[0], "in the first position.")
print(message, " has ",message[11], "in the last position.")

ตัวอย่างที่ 5 สร้างลิสต์จากสตริง
list(message)
strlist = list(message)
print("List generated from string", strlist)
print("Member at the first position:",strlist[0])
print(tuple(message))

EP.6 ชนิดข้อมูล Dictionary

Dictionary
แสดงกลุ่มข้อมูลที่สามารถกำหนดในลักษณะ Key-value คล้าย Associative array

ตัวอย่างที่ 1 การกำหนดตัวแปรที่มีชนิดข้อมูล Dictionary (dict)
book ={ "title":"Basic python","price": 200, "published":"23/03/2022" }
print(book)
print("Title: ",book["title"])
print("Price: ",book["price"])
print("Published:",book["published"])

ตัวอย่างที่ 2 การแก้ไขค่าใน Dictionary 
book["price"] = 500
print(book) 

ตัวอย่างที่ 3 การลบสมาชิกใน Dictionary
del(book["published"])
print(book)

ตัวอย่างที่ 4 สร้างลิสต์จาก Key ของ Dictionary
newlist = list(book)
print(newlist)

ตัวอย่างที่ 5 การสร้างลิสต์ที่เกิดจากการเรียงลำดับคีย์ของ Dictionary
newsort = sorted(book)
print(newsort)

ตัวอย่างที่ 6 การค้นหาค่าคีย์ใน Dictionary
'price' in book 
'published' in book
'published' not in book

EP.8 คำสั่งควบคุม if...else, if...elif..else

คำสั่งควบคุม

มี 3 กลุ่ม

1. คำสั่งควบคุมแบบสองทางเลือก. if...else, if...elif...else
2. คำสั่งควบคุมแบบหลายทางเลือก switch...case
3. คำสั่งควบคุมแบบวนซ้ำ for...in..., while... 

ตัวอย่างที่ 1 คำสั่งควบคุมแบบสองทางเลือก if...else
x = 50
if (x > 50):
   print("Variable is bigger than 50.")
   print("Variable x is ",x)
else: #x <= 50
   print("Variable may be 50 or less than 50.")
   print("Variable x is ",x)

ตัวอย่างที่ 2 คำสั่ง if..elif..else
score = 80
if (score >= 80):
   print("Got A!")
elif (score >=70):
   print("Got B!")
elif (score >=60):
   print("Got C!")
elif (score >=50):
   print("Got D!")
else:
   print("Got F!")

EP.9 สร้างคำสั่งแบบหลายทางเลือกด้วย if..elif..else และ dict

คำสั่งควบคุมแบบหลายทางเลือกใน Python

ในภาษา Python ไม่มีคำสั่ง switch...case แต่สามารถใช้ if...elif..else แทนได้ หรือใช้ชนิดข้อมูลแบบ dict

ตัวอย่างภาษา C++ ที่มี switch...case

#include <iostream>

using namespace std;

int main () {

int day = 1;

switch(day) {

case 1 :
cout << "Monday" << endl; break;
case 2 :
cout << "Tuesday" << endl;break;
case 3 :
cout << "Wednesday" << endl;break;
case 4 :
cout << "Thursday" << endl;break;
case 5 :
cout << "Friday" << endl;break;
case 6 :
cout << "Saturday" << endl;break;
case 7 :
cout << "Sunday" << endl;break;
default :
cout << "Unknow" << endl;
}

return 0;

}

EP.10 การใช้คำสั่งวนซ้ำ for...in และ while

คำสั่งวนซ้ำ

for
ตัวอย่างที่ 1 การใช้คำสั่ง for...in เพื่อทำงานกับลิสต์
fruit = ['apple', 'mango', 'pineapple','watemelon']
for name in fruit:
print(name)

ตัวอย่างที่ 2 การใช้คำสั่ง for...in กับ tuple
mytuple = ('Hi', 200, ["A", "B"])
for myinfo in mytuple:
   print(myinfo)

ตัวอย่างที่ 3 การใช้คำสั่ง for...in กับ dictionary
book = { "name" : "Jonathan", "price": 200 }
for mybook in book:
   print(mybook," has value:", book[mybook])

ตัวอย่างที่ 4 การใช้คำสั่ง for...in range
for number in range(100): #number 0...99
   print(number)

for number in range(5,10): #start 5 ...9
   print(number)

คำสั่ง while

ตัวอย่างที่ 1 การใช้คำสั่ง while มีนิพจน์
x = 0
while (x <=5):    #start x = 0 end at x = 5
   print(x)
   x += 1               # x = x + 1

ตัวอย่างที่ 2 การใช้ else ร่วมกับการวนซ้ำ 
x = 0
while (x <=5):   #start x = 0 end at x = 5
   print(x)
   x += 1              # x = x + 1
else:
   print("Else:", x)

EP.11 การสร้างฟังก์ชัน

ฟังก์ชัน

ตัวอย่างที่ 1 การประกาศฟังก์ชัน
def sumcalculator(x,y):
   result = x + y
   print(result)
   return result

sumtotal = sumcalculator(12,15)
print(sumtotal)

ตัวอย่างที่ 2 การประกาศฟังก์ชันที่มีพารามิเตอร์มีค่าดีฟอลต์
def sumcalculatorwithdefault(x=0, y =0):
   result = x + y
   print(result)
   return result

sumtotal = sumcalculatorwithdefault(12,15)
print("Result", sumtotal)
sumtotal = sumcalculatorwithdefault()
print("Result", sumtotal)
sumtotal = sumcalculatorwithdefault(12)
print("Result", sumtotal)

ตัวอย่างที่ 3 การจับคู่ของพารามิเตอร์ โดยการระบุชื่อพารามิเตอร์ จะมีการกำหนดโดยใช้ชื่อ และไม่สนใจลำดับ
sumtotal = sumcalculatorwithdefault( y = 50,x = 40)
print(sumtotal)
sumtotal = sumcalculatorwithdefault( x = 40,y = 50)
print(sumtotal)
xvar = 80
yvar = 90
sumtotal = sumcalculatorwithdefault( x = xvar,y = yvar)
print(sumtotal)

EP.12 การรับอินพุตเข้ากับฟังก์ชัน input()

ฟังก์ชัน Input

ใช้ในการรับข้อมูลจากคีย์บอร์ด
print("Please enter price: ")
x = input("Price:")
print(x)

ปกติฟังก์ชัน input() จะรับค่าข้อมูลเข้าซึ่งเป็น String 
a = input("number1:")
b = input("number2:")
result = a + b # ตัวดำเนินการ + เป็นการต่อสตริง
print("Sum is ",result," ",type(result))              # ผลลัพธ์เป็นสตริง
result = int(a) + int(b)                                         # ผลลัพธ์เป็นเลขจำนวนเต็ม
print("Sum is ",result," ",type(result))
